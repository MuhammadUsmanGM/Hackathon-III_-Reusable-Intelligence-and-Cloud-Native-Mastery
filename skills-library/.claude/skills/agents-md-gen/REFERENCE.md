# AGENTS.md Generator - Reference

## Purpose
The AGENTS.md file describes a repository's structure, conventions, and guidelines so AI agents can understand how to work with the codebase effectively.

## Standard Sections
- **Repository Structure**: Directory layout and important files
- **Coding Standards**: Language-specific conventions and style guides
- **Architecture Patterns**: Frameworks, libraries, and architectural decisions
- **Testing Guidelines**: How to run tests and expected coverage
- **Deployment Process**: Steps to build and deploy the application
- **Troubleshooting**: Common issues and solutions

## Generation Options
- `--template`: Use a specific template (basic, advanced, enterprise)
- `--include-tests`: Include test-related sections
- `--custom-sections`: Add custom sections based on repository content
