# AGENTS.md Generator

## When to Use
- Creating AGENTS.md files for new repositories
- Updating existing AGENTS.md with new patterns
- Teaching AI agents about repository structure

## Instructions
1. Run the generator script: `./scripts/generate_agents_md.py <repo_path>`
2. The script will analyze the repository structure
3. Creates or updates AGENTS.md with appropriate sections

## Validation
- [ ] AGENTS.md file created successfully
- [ ] Contains appropriate repository sections
- [ ] Follows AAIF standards

See [REFERENCE.md](./REFERENCE.md) for configuration options.
