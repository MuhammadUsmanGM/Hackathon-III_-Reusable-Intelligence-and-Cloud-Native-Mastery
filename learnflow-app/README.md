# LearnFlow Application

An AI-powered Python tutoring platform built using agentic infrastructure with Claude Code and Goose.

## Architecture

- **Frontend**: Next.js with Monaco editor
- **Backend**: FastAPI microservices with Dapr
- **Database**: PostgreSQL (Neon)
- **Messaging**: Apache Kafka on Kubernetes
- **AI Agents**: Multi-agent system for tutoring (Triage, Concepts, Code Review, Debug, Exercise, Progress)

## Purpose

This application demonstrates the power of reusable skills by being built entirely through AI agents using the skills in the skills-library repository.
